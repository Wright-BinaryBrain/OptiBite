## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

`npm install`

To Run the server:

`npm run dev`

To Visit the server:

`localhost:4000\api\v1`

To Visit Docs

`localhost:4000\api\docs`
